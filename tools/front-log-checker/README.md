# 前端 LOG 驗證（front-log-checker）

此工具提供 `intercept.js`，用於在瀏覽器頁面攔截並檢查前端 LOG。

## 使用方式

1. 在 QA Tools Hub 切到「前端 LOG 驗證」。
2. 點「複製」取得完整 `intercept.js`。
3. 到目標網站打開 DevTools Console。
4. 貼上整段腳本並執行。
5. 依流程操作頁面，觀察輸出是否符合規則。
6. 可使用面板按鈕導出：
   - `匯出 CSV`：人工查看欄位值
   - `匯出 JSON(原始)`：完整攔截資料
   - 建議將 `JSON(原始)` 匯入比對工具，選擇匹配鍵與比對項目後再做差異分析

## 注意事項

- 建議在測試環境使用，避免在正式站長時間掛載攔截。
- 每次測試前可重新貼上腳本，確保使用最新版本。
