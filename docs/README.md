# Playwright 測試項目 - 多文件模組化版本

這是一個重構後的 Playwright 自動化測試項目，採用多文件模組化架構，便於維護和擴展。

## 📚 文檔索引

### 測試標準文檔（必讀）
- **[TEST_SPECIFICATION.md](./TEST_SPECIFICATION.md)** - 測試規格文檔：定義測試範圍、業務規則、驗證點
- **[TEST_STANDARDS.md](./TEST_STANDARDS.md)** - 測試標準與規範：測試分類、命名規範、代碼標準
- **[TEST_SCENARIO_MATRIX.md](./TEST_SCENARIO_MATRIX.md)** - 測試場景矩陣：所有測試場景的完整列表和狀態

### 技術文檔
- [CHANGELOG.md](./CHANGELOG.md) - 變更日誌
- [QUICK_START.md](./QUICK_START.md) - 快速開始指南
- [TROUBLESHOOTING.md](./TROUBLESHOOTING.md) - 故障排除指南
- [LOGGER_USAGE.md](./LOGGER_USAGE.md) - 日誌工具使用指南
- [REFACTORING_SUMMARY.md](./REFACTORING_SUMMARY.md) - 重構總結
- [CHIP_SCROLLING_FIX.md](./CHIP_SCROLLING_FIX.md) - 籌碼滾動修復記錄
- [ORIGINROUNDCODE_IMPROVEMENT.md](./ORIGINROUNDCODE_IMPROVEMENT.md) - 局號改進記錄
